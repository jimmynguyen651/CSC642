Frequently Asked Questions

How to Review Resturants

Read guidelines to reviewing a resturant.

  • Under no circumstances should you consult the restaurant or have any deals/relations with the business when reviewing any restaurant.
  • Read each item carefully.
  • Verify each item carefully.
  • IF each category is approved, you must APPROVE the listing.
  • IF one or more categories is denied, you MUST decline the listing.
How to Review Flagged Reviews

Read guidelines to reviewing flagged reviews.

  • Under no circumstances should you consult any users associated with the report.
  • Read the flagged review carefully.
  • Verify if flagged review breaks Tidbits guidelines.
  • If flagged review does not break Tidbits guidelines, do NOT delete review.
How to Review User Accounts

Read guidelines to reviewing user accounts.

  • Under no circumstances should you consult the user with the account.
  • Read their information carefully.
  • Read any reports about the user and verify the report.
